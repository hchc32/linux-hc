1.函数是属于类的,不是属于对象的

2.构造函数(可以给参数) -- 对象被创建的时候调用
构造函数的名字和类的名字是相同的,没有返回类型

    class X
    {
        int i;
    public:
        x();
    };
    构造函数会在对象被创建的时候自动调用
    new时,先分配空间,然后调用构造函数

3.析构函数(不可以传参数) --  当对象被消灭的时候调用的
    delete时,先析构,然后释放空间

4.int *psome = new int [10];
  delete [] psome;

5.public和private
    private是对类来说的，同一个类的不同对象之间的私有成员是可以访问的
 
6. ::是范围解析运算符

7.this指针是访问类的地址的

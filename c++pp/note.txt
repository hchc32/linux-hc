1.函数是属于类的,不是属于对象的

2.构造函数(可以给参数) -- 对象被创建的时候调用
构造函数的名字和类的名字是相同的,没有返回类型

    class X
    {
        int i;
    public:
        x();
    };
    构造函数会在对象被创建的时候自动调用
    new时,先分配空间,然后调用构造函数

3.析构函数(不可以传参数) --  当对象被消灭的时候调用的
    delete时,先析构,然后释放空间

4.int *psome = new int [10];
  delete [] psome;

5.public和private
    private是对类来说的，同一个类的不同对象之间的私有成员是可以访问的
 
6. ::是范围解析运算符

7.this指针是访问类的地址的

8.继承
    子类在父类的基础上增加的；
    父类中的private在子类中是存在的,但是不可以碰;
    protected: 受保护的对象，只能在本身和子类中用;
    
9.istream流
  效果是检测流的状态，流遇到未知错误,istream对象的状态就会变为无效

10.getline函数,丢掉了后面的换行符号.

11.操作string的每一个字符,需要用到字符串的引用,引用是给字符串一个别名

12.要想使用vector
    #include<vector>
    using std::vector;
    vector是一个模板,而不是类型,vector内容纳的是对象,而引用不是对象,所以没有包含引用的vector

13.vector不可以用下标添加元素

14.迭代器的类型
    iterator         可读写
    const_iterator   只读
